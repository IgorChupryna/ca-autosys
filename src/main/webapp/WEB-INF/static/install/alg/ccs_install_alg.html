Installing on UNIX

(UNIX) Install Common Components Using Console Mode

This procedure describes the steps to install the common components on UNIX using console mode. As you proceed through the installation, note that pressing the Tab key moves the cursor to a field or an action in a panel. Pressing the spacebar selects or clears an option.

Follow these steps:

  1. Ensure that the $DISPLAY environment variable is unset.
    For example, enter the following:

    unset DISPLAY
  2. Mount the CA Common Components media.

  3. Run the following command:

    ./ccc_setup.sh

    Note: The setup script performs various system checks, which takes few minutes.
    The Welcome message appears with Next selected, after the system checks are complete.

  4. Press Enter.
    The License Agreement panel appears.

  5. Press the plus key (+) to scroll through the page and read the license text and select I Agree.
    The Component Selection panel appears.

  6. Select the components you want to install by pressing the down-arrow key (or Tab key) to move the cursor through the list of components, and using the spacebar to select or clear an item.
  7. Select Next.
    The next panel appears depending on the components that you selected.
  8. Continue with the installation by entering the required information on each panel and selecting Next.
    The Review Settings panel appears after the last data entry panel, listing the components that you selected to install.

  9. Review the information and, if it is correct, select Next.

    Note: To make a change to an entry, select Back as many times as necessary to locate that entry. Then, make the appropriate change, select Next until the Review Settings panel appears, and select Next.

    The installation process begins and the progress is displayed. When the installation completes, you are prompted to exit the installation.

  10. Select Finish.
  11. Log out and then log back in to reset the environment.
    The common components installation is complete.
